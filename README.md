javaadapterpattern
==================

Java Adapter Pattern

┌───────────────┐              ┌───────────────┐                ┌──────────────────┐
│     Client    │─────────────>│     Target    │         ┌─────>│      Adaptee     │
└───────────────┘              ├───────────────┤         │      ├──────────────────┤
                               │    Request()  │         │      │ SpecificRequest()│
                               └───────────────┘         │      └──────────────────┘
                                       ▲                 │
                                       │                 │
                                       │                 │
                               ┌───────────────┐ adaptee │
                               │    Adapter    │─────────┘
                               ├───────────────┤                ┌────────────────────────────┐
                               │    Request()○-│- - - - - - - - ┤ adaptee->SpecificRequest() │
                               └───────────────┘                └────────────────────────────┘
